## Start here

If you're reading this, you probably have an idea — or you have an MVP and no users. Maybe both. Maybe you've sketched it on napkins. Maybe you've already paid someone $5,000 for a "consultation" that ended in a 40-slide deck and no product. Maybe you launched and nothing happened.

This library is for you.

It's organized as a **sequential journey** in 4 phases, ending in a very specific place: a working SaaS, paying customers, and live distribution channels bringing more of them in. **In 2026, building is 5% of the game. Distribution is 95%.** This library covers both.

## Who this is for

- **Non-technical founders** who can't write code (yet) but have a real idea worth shipping
- **Aspiring solo SaaS founders** who want a tactical, no-BS path to revenue *and* growth
- **First-time technical founders** who can ship code but freeze when it's time to talk to customers or post on Reddit
- **Founders stuck post-launch** with $0–$1k MRR who are wondering why "great products don't sell themselves"
- **Anyone tired of startup advice that doesn't translate into "what do I do tomorrow morning"**

If you're looking for a guide to raising a Series A, this isn't that. If you're looking for hiring advice, also not that. This is **idea → product → revenue → distribution.** The first 12–18 months of your business — the part that decides whether the next 10 years are even possible.

## Who this is *not* for

- People who want to "explore startup ideas" indefinitely
- Founders looking for technical depth — this library tells you *what* to build, *who* should build it, and *how* to distribute it; not how to write the code yourself
- Anyone who wants permission to skip distribution
- People building purely for fun (this is a revenue-focused playbook)

## How to actually use this library

### Path A — The full founder track (12–18 months)

Read it sequentially. Do every "Your Turn" exercise. Treat each chapter as a real assignment. At the end of 12–18 months, you'll have:

- A validated idea (Phase 1)
- A scoped MVP that ships (Phase 2)
- First paying customers (Phase 3)
- Two distribution channels producing inbound traffic (Phase 4)
- Real, repeatable revenue between $5k–$50k MRR

### Path B — The reference track

Already shipping? Use the chapters as a reference. Stuck on pricing? Jump to Chapter 8. About to start LinkedIn? Chapter 17. About to attempt programmatic SEO? Chapter 14. Bookmark this repo. Come back when you hit a wall.

### Path C — Specific phase

If you're already past the build phase, skip to Phase 4 (Chapters 11–20). If you've already built and have customers but no distribution, **Phase 4 is the most valuable phase of the library for you.**
